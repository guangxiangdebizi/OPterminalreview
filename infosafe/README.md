### 密码学部分（⭐⭐⭐⭐⭐）

1. **DES参数**：64位分组、56位密钥、16轮、Feistel结构
2. **S盒使用**：6位输入→4位输出（**输出必须是二进制！**）
3. **雪崩效应**：改变1位，约50%密文位变化
4. **RSA算法**：
   - 完整实现步骤
   - 为什么基于大整数因式分解
   - **为什么不能抵抗选择明文攻击**（必考）

   - 计算实例

3. **数字签名 vs 消息鉴别码**：不可抵赖性是关键
4. **数字证书**：本质就是数字签名

### 网络安全部分（⭐⭐⭐⭐⭐）

   - **横向隔离**（物理隔离、网闸）

## 📝 考试注意事项

### 答题技巧

**1. 不定项选择题**：
- 可能单选或多选
- 评分严格（错一个可能全扣）
- 开卷也要认真做

**2. 填空题**：
- 基础知识考查
- 提前熟悉知识点位置
- 可以快速查找

**3. 简答题**：
- **需要理解，不能纯抄书**
- 要用自己的话表达
- 注意使用书面语

**4. 应用题**：
- DES的S盒使用（**输出必须是二进制**）
- RSA计算题
- 提前练习，考场来不及现算

**5. 综合题**：
- 课堂讲过但课本没有（如电力系统安全）
- 需要深入理解（如震网病毒）
- **要体现主动学习和思考**
- 开放性问题要答到点子上

### 老师强调的易错点

1. **S盒输出**："输出必须是二进制形式（如1001），不能写十进制（如9）"
2. **BLP模型**："是上读下写，不要写成下读上写"
3. **RSA攻击**："不能抵抗选择明文攻击，每年都有人答错"
4. **震网病毒**："不能只从书上抄，匹配性不高只给2-3分"
